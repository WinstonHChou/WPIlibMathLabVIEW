PathPlanner.lvlib:PathPlanner_Util_ReadJson.vi

Imports the Path Planner WayPoints from a JSON file.

Parameters:

- FileName - File name string of the json file to import from

- Error In - Error cluster (not used)

Returns:

- Trajectory - Trajectory data structure

- Error out - Returned error cluster

Note:

-- Default path for roboRIO is /home/lvuser/natinst/LabVIEW Data

-- Default path for Windows is: %USERPROFILE%\Documents\LabVIEW Data
