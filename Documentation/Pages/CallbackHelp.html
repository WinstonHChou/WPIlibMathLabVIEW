CallbackHelp

CallbackHelp_MatrixMinus

matrix callback function that performs



      A - B



Inputs:

   - ExtraData  -- Variant containing extra data -- not used

   - A  -- A matrix

   - B  --  B matrix



Outputs:

   - Output  -- Resulting matrix

   - Error  --  If TRUE, an error occured.
