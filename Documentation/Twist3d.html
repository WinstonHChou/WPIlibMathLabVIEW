Trajectory_Library.lvlib:Twist3d_Create.vi

Constructs a Twist3d with the given values.

Parameters:

- dx - Change in x direction relative to robot.

- dy - Change in y direction relative to robot.

- dz - Change in z direction relative to robot.

- rx - Rotation vector x component.

- ry - Rotation vector y component.

- rz - Rotation vector z component.

Results:

- TWIST3D - TWIST3D data structure

 dX

Linear "dx" component.

dY

Linear "dy" component.

dZ

Angular "dtheta" component (radians).

rX

Linear "dx" component.

rY

Linear "dy" component.

rZ

Angular "dtheta" component (radians).

Twist3d

Twist2D data structure typdef. A change in distance along arc since the last pose update. We can use ideas from differential calculus to create new Pose2ds from a Twist2d and vise versa.

A Twist can be used to represent a difference between two poses.
